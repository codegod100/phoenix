# Dashboard Archive UI

Archive tab interface and active/archived view switching.

## Archive Tab Navigation

- Archived tasks must be viewable via a separate "Archived Tasks" tab or filter
- Switching between Active and Archived views must update the task list without a full page reload

## Archive Data

- Archived tasks must be queryable separately and restorable to active status
- Users must be able to archive completed tasks (hide from active views but retain data)
- Archived status must be visually indicated on the task card status badge with a muted/dim appearance

## Archive Status Badge

- Archived tasks must display an "archived" status badge when viewing the Active tab (dimmed/strikethrough style)
- When viewing the Archived tab, tasks show their original status with an archived indicator overlay

## Archive Display

- Archived tasks must be displayed in the same grid layout as active tasks
- Archived task cards must have a "Restore" button to reactivate them
