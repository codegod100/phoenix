# Phoenix Drift

Detect drift between specification and implementation.

## When to Use

- To check if generated files match manifest
- After editing specs to see impact
- To find unauthorized code modifications
- Before committing to verify state
- To trace requirements to implementation

## Usage

```bash
/skill:phoenix-drift             # Full drift report
/skill:phoenix-drift --summary   # Brief overview
/skill:phoenix-drift --fix       # Auto-regenerate drifted files
/skill:phoenix-drift --trace REQ # Trace specific requirement
```

## Types of Drift

1. **Drifted**: File content doesn't match manifest hash
   - Spec changed and needs regeneration
   - Hand-edited generated code
   
2. **Orphan**: File exists but no IU owns it
   - Generated by old IU that's been removed
   - Manual file creation in wrong place
   
3. **Missing**: IU expects file but doesn't exist
   - First run or deleted file
   - Generation failed
   
4. **Clean**: File matches manifest

## What It Does

1. **Load manifest** (expected state)
2. **Scan files** (actual state)
3. **Compare hashes**
4. **Check traceability** (canon_ids → iu_id → file)
5. **Analyze changes** (spec changes causing drift)
6. **Report findings** with recommended actions

## Implementation Steps

### Step 1: Load Data

```typescript
const { root, phoenixDir } = requireProjectRoot();
const manifest = loadManifest(phoenixDir);
const iuGraph = readGraph<IUGraph>(phoenixDir, 'ius');
const canonGraph = readGraph<CanonicalGraph>(phoenixDir, 'canonical');
```

### Step 2: Check Each Manifest Entry

```typescript
const driftedFiles: DriftedFile[] = [];
const syncFiles: SyncFile[] = [];

for (const [filePath, entry] of Object.entries(manifest?.files || {})) {
  const fullPath = join(root, filePath);
  
  // Check file exists
  if (!existsSync(fullPath)) {
    driftedFiles.push({
      path: filePath,
      expectedHash: entry.hash,
      actualHash: null,
      status: 'missing',
      iuId: entry.iu_id
    });
    continue;
  }
  
  // Compute current hash
  const currentHash = computeFileHash(fullPath);
  
  if (currentHash === entry.hash) {
    syncFiles.push({
      path: filePath,
      hash: currentHash,
      status: 'sync',
      iuId: entry.iu_id
    });
  } else {
    driftedFiles.push({
      path: filePath,
      expectedHash: entry.hash,
      actualHash: currentHash,
      status: 'drift',
      iuId: entry.iu_id
    });
  }
}
```

### Step 3: Find Orphan Files

```typescript
const orphanFiles: string[] = [];
const generatedDir = join(root, 'src', 'generated', 'app');

if (existsSync(generatedDir)) {
  const allFiles = findAllFiles(generatedDir);
  const manifestFiles = new Set(Object.keys(manifest?.files || {}));
  
  for (const file of allFiles) {
    const relPath = relative(root, file);
    if (!manifestFiles.has(relPath)) {
      orphanFiles.push(relPath);
    }
  }
}
```

### Step 4: Analyze Drift Causes

```typescript
function analyzeDrift(
  file: DriftedFile,
  iuGraph: IUGraph,
  canonGraph: CanonicalGraph
): DriftAnalysis {
  const iu = iuGraph?.ius?.find(i => i.iu_id === file.iuId);
  
  if (!iu) {
    return { type: 'unknown_iu', explanation: 'IU not found in graph' };
  }
  
  // Read current file
  const content = readFileSync(join(root, file.path), 'utf8');
  
  // Check _phoenix metadata
  const phoenixMatch = content.match(/export const _phoenix = \{[\s\S]*?\}/);
  if (!phoenixMatch) {
    return { 
      type: 'missing_metadata', 
      explanation: 'File missing _phoenix export - likely hand-created' 
    };
  }
  
  // Check iu_id match
  const idMatch = phoenixMatch[0].match(/iu_id:\s*['"]([^'"]+)['"]/);
  if (!idMatch || idMatch[1] !== iu.iu_id) {
    return {
      type: 'wrong_iu',
      explanation: `File has wrong IU ID (${idMatch?.[1]?.slice(0, 8)}... vs ${iu.iu_id.slice(0, 8)}...)`
    };
  }
  
  // Check canon_ids coverage
  const canonIdsMatch = phoenixMatch[0].match(/canon_ids:\s*\[([^\]]*)\]/);
  const fileCanonIds = canonIdsMatch 
    ? canonIdsMatch[1].split(',').map(s => s.trim().replace(/['"]/g, '')).filter(Boolean)
    : [];
  
  const missingCanonIds = iu.source_canon_ids.filter(id => !fileCanonIds.includes(id));
  if (missingCanonIds.length > 0) {
    return {
      type: 'incomplete_coverage',
      explanation: `Missing ${missingCanonIds.length} canonical requirements`,
      details: missingCanonIds
    };
  }
  
  // Check if spec changed
  // (Would need to compare with warm hashes)
  
  return {
    type: 'unknown_change',
    explanation: 'Content changed but metadata intact - manual edit or regeneration needed'
  };
}
```

### Step 5: Generate Report

```typescript
const report: DriftReport = {
  status: driftedFiles.length === 0 && orphanFiles.length === 0 ? 'clean' : 'drifted',
  summary: {
    totalFiles: Object.keys(manifest?.files || {}).length + orphanFiles.length,
    sync: syncFiles.length,
    drifted: driftedFiles.length,
    missing: driftedFiles.filter(f => f.status === 'missing').length,
    orphan: orphanFiles.length
  },
  driftedFiles: driftedFiles.map(f => ({
    ...f,
    analysis: analyzeDrift(f, iuGraph!, canonGraph!)
  })),
  orphanFiles,
  iusAffected: [...new Set(driftedFiles.map(f => f.iuId))].filter(Boolean)
};
```

### Step 7: Trace Requirements (Optional)

For `--trace` mode:

```typescript
function traceRequirement(canonId: string, canonGraph: CanonicalGraph, iuGraph: IUGraph) {
  const node = canonGraph.nodes[canonId];
  if (!node) return { error: 'Canon node not found' };
  
  // Find IU that implements this
  const implementingIU = iuGraph.ius.find(iu => 
    iu.source_canon_ids.includes(canonId)
  );
  
  // Find generated file
  const files = implementingIU?.output_files || [];
  
  return {
    requirement: node.statement,
    canonId: node.canon_id,
    confidence: node.confidence,
    iu: implementingIU ? {
      id: implementingIU.iu_id,
      name: implementingIU.name,
      contract: implementingIU.contract.description
    } : null,
    files,
    linkedRequirements: node.linked_canon_ids.map(id => ({
      id,
      statement: canonGraph.nodes[id]?.statement,
      type: node.link_types[id]
    }))
  };
}
```

### Step 8: Handle Fixes

For `--fix` mode:

```typescript
if (args.includes('--fix')) {
  for (const drift of report.driftedFiles) {
    if (drift.analysis.type === 'incomplete_coverage' || 
        drift.analysis.type === 'unknown_change') {
      // Regenerate this IU
      const iu = iuGraph?.ius.find(i => i.iu_id === drift.iuId);
      if (iu) {
        await regenerateIU(iu, context);
        console.log(`  Regenerated ${iu.name}`);
      }
    }
  }
  
  for (const orphan of report.orphanFiles) {
    // Ask user or auto-delete
    console.log(`  Orphan: ${orphan} (manual review needed)`);
  }
}
```

## Drift Justification

If manually editing generated files (emergency only):

```typescript
// Create drift justification
const justification = {
  timestamp: new Date().toISOString(),
  file: filePath,
  reason: 'Emergency fix for production bug',
  expected_regen: 'Within 48 hours',
  canon_nodes_to_add: ['new-requirement-id']
};

writeFileSync(
  join(phoenixDir, '.pi-agent', 'drift-justification.md'),
  JSON.stringify(justification, null, 2)
);
```

Mark in code:
```typescript
// DRIFT: temporary_patch - Production hotfix for #123
// Regenerate planned: 2026-04-08
// Spec update needed: retry logic for network failures
```
